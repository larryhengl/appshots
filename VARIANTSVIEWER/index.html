Snaps of apps I built internally at Novartis for informatics research workflows. Scrubbed for our (in)sanity.

Goals: Variants Viewer is a BioMarin web app used for mining genomic mutations in rare diseases, and calculating disease incidence estimates in geo-ethnic populations.

Notes: This app was built using React, React-Router, D3 for data viz, dc.js for Crossfilter facets, FixedDataTable, Baobab for state tree management, jStat, Bootstrap/FontAwesome, Gulp + Babel for app builds, and various supporting js libs.
